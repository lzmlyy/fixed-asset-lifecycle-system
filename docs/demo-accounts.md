## 四、演示建议

1. **完整流程演示**：用 `system.manager` 登录，展示全系统功能。
2. **岗位分工演示**：依次用 `asset.manager` → `dept.leader` → `finance.officer` → `audit.officer` 登录，展示不同角色看到的菜单和首页差异。
3. **权限隔离演示**：用 `office.staff` 登录后尝试访问折旧报表 URL，会跳转 403 页面。
4. **盘点闭环演示**：用 `inventory.clerk` 登录，展示盘点任务执行流程。
