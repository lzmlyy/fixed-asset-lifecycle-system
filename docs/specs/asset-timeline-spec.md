# 资产生命周期时间线规格草案

> 状态：规划中
> 对应 SDD 草案：`docs/sdd/phase-7-asset-timeline-draft.md`

## 1. 功能定义

在资产详情页展示资产从入库、领用、调拨、维修、报废、审批到操作日志的完整记录，以时间线形式呈现。

## 2. 做什么

- 整合生命周期单据（入库、领用、调拨、维修、报废）
- 整合审批记录（提交、通过、驳回）
- 整合操作日志（状态变更）
- 时间线按时间倒序展示
- 支持按事件类型筛选

## 3. 不做什么

- 不修改已有业务表结构
- 不做时间线事件导出
- 不做时间线事件编辑
- 时间线为只读聚合视图

## 4. 数据来源

| 事件类型 | 数据来源 |
|---|---|
| 入库 | `asset_inbound_order` |
| 领用 | `asset_receive_order` |
| 调拨 | `asset_transfer_order` |
| 维修 | `asset_repair_order` |
| 报废 | `asset_scrap_order` |
| 审批 | `approval_record` |
| 操作日志 | `asset_operation_log` |

## 5. 接口规划

- `GET /api/asset/{id}/timeline` - 获取资产时间线
  - 查询参数：`eventType`（可选，筛选事件类型）
  - 返回：时间线事件列表（按时间倒序）

## 6. 前端页面

- 资产详情页增加时间线区域
- 新增 `AssetTimeline.vue` 通用组件
- 支持事件类型筛选标签

## 7. 验收标准

- 资产详情页可展示完整时间线
- 时间线包含所有事件类型
- 时间线按时间倒序排列
- 可按事件类型筛选
- 时间线为只读
