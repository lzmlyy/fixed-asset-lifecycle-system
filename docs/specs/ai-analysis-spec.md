# AI 辅助分析规格草案

> 状态：规划中
> 对应 SDD 草案：`docs/sdd/phase-8-ai-analysis-draft.md`

## 1. 功能定义

基于系统内部数据生成资产状态摘要、异常资产提示、维修/报废辅助建议和管理报告辅助内容。AI 输出只作为辅助参考，不直接修改业务数据。

## 2. 做什么

- 资产状态摘要（资产总数、状态分布、关键指标）
- 异常资产提示（长期闲置、维修频繁、状态异常）
- 维修/报废辅助建议（基于维修次数、使用年限、净值）
- 管理报告辅助生成（汇总统计文本）

## 3. 不做什么

- AI 输出不直接修改业务数据
- 不接入外部 AI 服务或大模型 API
- 不做自然语言对话交互
- 不处理图像、视频或文档文件
- 不做预测性分析（仅基于现有数据统计）

## 4. 数据来源

| 数据 | 来源 |
|---|---|
| 资产基础数据 | `asset` 表 |
| 维修记录 | `asset_repair_order` 表 |
| 报废记录 | `asset_scrap_order` 表 |
| 审批记录 | `approval_record` 表 |
| 操作日志 | `asset_operation_log` 表 |

## 5. 接口规划

- `GET /api/ai/summary` - 资产状态摘要
- `GET /api/ai/alerts` - 异常资产提示
- `GET /api/ai/suggestions` - 维修/报废辅助建议
- `GET /api/ai/report` - 管理报告辅助内容

## 6. 前端页面

- AI 辅助分析页（已有骨架，待完善）
  - 资产状态摘要区域
  - 异常资产提示区域
  - 维修/报废辅助建议区域
  - 管理报告辅助生成区域

## 7. 验收标准

- 可展示资产状态摘要
- 可展示异常资产提示
- 可展示维修/报废辅助建议
- 可生成管理报告辅助内容
- AI 输出明确标注为"辅助参考"
- AI 输出不修改任何业务数据
