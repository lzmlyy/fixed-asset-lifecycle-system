## 二、逐项验收明细

### 1. 登录鉴权

| 验收项 | 接口/页面 | 验收结果 | 说明 |
|---|---|---|---|
| 登录接口 | `POST /api/auth/login` | 通过 | 校验用户名密码（BCrypt），签发 JWT Token，返回用户信息与权限列表 |
| JWT Token | `JwtUtil` + `AuthInterceptor` | 通过 | Token 有效期 720 分钟，拦截器统一解析并写入 `UserContext` |
| 路由守卫 | `router.beforeEach` | 通过 | 未登录跳转 `/login`，已登录访问 `/login` 重定向到 `/dashboard` |
| 401 跳转 | axios 响应拦截 | 通过 | Token 失效或过期时前端自动清空登录态并跳转登录页 |
| v-permission 指令 | 前端全局指令 | 通过 | 按钮级权限控制生效，无权限元素被移除 |
