# 功能设计

## 1. 数据库设计

> 新增哪些表？修改哪些表？字段定义、索引、约束。

## 2. 后端包结构

> 新增的包路径和说明。

## 3. Entity 设计

> Entity 类名、字段、注解、表名映射。

## 4. DTO 设计

> 请求 DTO 类名、字段、校验规则。

## 5. VO 设计

> 响应 VO 类名、字段、数据来源。

## 6. Mapper 设计

> Mapper 接口、基础方法、自定义 SQL。

## 7. Service 设计

> Service 方法签名、业务逻辑说明、事务边界。

## 8. Controller 设计

> Controller 类名、请求方法、路径、参数。

## 9. 前端 API 设计

> API 文件、函数名、调用路径。

## 10. 前端页面设计

> 页面路径、组件结构、表单设计、表格设计、弹窗设计。

## 11. 状态流转设计

> 状态枚举、允许的转换、每个转换触发的业务逻辑。

## 12. 权限设计

> 哪些接口需要哪些角色才能访问。

## 13. 日志设计

> 哪些操作需要写入操作日志？写入什么内容？
