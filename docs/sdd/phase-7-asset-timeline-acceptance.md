## 2. 功能验收

### 2.1 资产详情页基础信息不变

- 登录 admin / 123456
- 进入资产台账列表
- 点击任意资产查看详情
- 确认资产编号、名称、分类、品牌、规格、原值、净值、累计折旧、残值率、使用年限、折旧方法、购置日期、所属部门、使用人、存放地点、资产状态、备注均正常展示
- 确认返回按钮可用

### 2.2 生命周期时间线展示

- 资产详情页基础信息卡下方出现「生命周期时间线」区域
- 时间线按时间倒序排列（最新事件在前）
- 每条事件展示：标题、事件类型标签、时间、操作人、描述
- 有状态变化的事件展示「状态变化：XXX → YYY」
- 有单据编号的事件展示「单据编号：XXX」
- 有备注的事件展示备注

### 2.3 事件类型筛选

- 右侧下拉框选项：全部、入库、领用、调拨、维修、报废、审批、操作日志
- 选择「全部」：显示所有事件
- 选择「入库」：只显示入库事件
- 选择「领用」：只显示领用事件
- 选择「调拨」：只显示调拨事件
- 选择「维修」：只显示维修事件
- 选择「报废」：只显示报废事件
- 选择「审批」：只显示审批事件
- 选择「操作日志」：只显示操作日志事件

### 2.4 空数据状态

- 找一个无生命周期单据和操作日志的资产
- 确认时间线区域显示「暂无生命周期记录」

### 2.5 新增单据后刷新

- 在生命周期页面创建一次入库/领用/调拨/维修/报废单据
- 回到对应资产详情页
- 确认时间线新增对应事件
