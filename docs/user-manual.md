# 用户使用手册

## 1. 系统概述

国企固定资产全生命周期管理系统是一个面向国企、事业单位、大型企业的固定资产管理平台。

当前第一阶段提供以下功能：

- 系统登录
- 首页驾驶舱（资产数据总览）
- 资产台账管理（增删改查）
- 资产详情查看
- 资产分类查看
- 其他功能入口（骨架页面）

---

## 2. 登录系统

### 2.1 访问系统

打开浏览器，访问：`http://localhost:3000`

### 2.2 登录步骤

1. 在登录页左侧可看到系统名称和功能简介
2. 在右侧表单输入用户名和密码
3. 点击「登录」按钮

### 2.3 默认账号

| 账号 | 密码 | 角色 |
|---|---|---|
| admin | 123456 | 系统管理员 |

### 2.4 注意事项

- 登录失败时页面会提示错误信息
- 长时间不操作 token 会过期，需要重新登录

---

## 3. 系统布局

登录后进入主界面，包含三个区域：

1. **左侧导航菜单**：深蓝色背景，包含所有功能入口
2. **顶部用户栏**：白色背景，显示当前页面标题和当前用户名，右侧有「退出登录」按钮
3. **主内容区**：灰白背景，展示当前功能页面

---

## 4. 首页驾驶舱

### 4.1 功能介绍

首页展示固定资产的核心统计数据，帮助用户快速了解资产概况。

### 4.2 查看内容

第一行展示四个核心指标卡片：

- **资产总数**：系统中有效资产的总数量
- **资产总原值**：所有资产的购置原值合计
- **累计折旧**：所有资产的累计折旧合计
- **资产净值**：所有资产的当前净值合计

第二行展示四个状态指标卡片：

- **使用中**：当前正在使用的资产数量
- **闲置**：当前闲置的资产数量
- **维修中**：正在维修的资产数量
- **待报废**：等待报废审批的资产数量

第三行展示三个统计图表：

- **资产分类分布图**：各分类的资产数量占比
- **部门资产金额排行**：各部门的资产原值排行
- **月度折旧趋势图**：近半年的折旧趋势

### 4.3 注意事项

- 所有统计数据来自系统实时计算
- 折旧趋势图第一阶段为示例数据

## 5. 资产台账管理

### 5.1 功能介绍

资产台账是系统的核心功能，支持对固定资产进行新增、查询、编辑、删除和查看详情。

### 5.2 查询资产

操作步骤：

1. 点击左侧菜单「资产管理 > 资产台账」
2. 在页面上方可设置查询条件：
   - 资产编号、资产名称、资产分类、所属部门
   - 使用人、存放地点、资产状态、购置日期范围
3. 点击「查询」按钮查看结果
4. 点击「重置」按钮清空查询条件

### 5.3 新增资产

操作步骤：

1. 在资产台账页面点击「新增资产」按钮
2. 在弹出的表单中填写以下信息：
   - 资产名称（必填）
   - 资产分类（必填）
   - 品牌、规格型号
   - 购置日期（必填）
   - 原值（必填）
   - 使用年限（必填）
   - 残值率（必填）
   - 所属部门（必填）
   - 使用人（必填）
   - 存放地点
   - 二维码、RFID、备注（可选）
3. 点击「确认」按钮提交

注意事项：

- 资产编号由系统自动生成，无需手动填写
- 累计折旧和净值由系统自动计算
- 新增后资产默认状态为「闲置」

### 5.4 编辑资产

操作步骤：

1. 在资产台账表格中找到需要编辑的资产
2. 点击操作列的「编辑」按钮
3. 在弹出的表单中修改需要变更的字段
4. 可以选择修改资产状态
5. 点击「确认」按钮保存

注意事项：

- 编辑时如果修改了状态，系统会自动记录操作日志
- 累计折旧和净值会根据修改后的信息重新计算

### 5.5 删除资产

操作步骤：

1. 在资产台账表格中找到需要删除的资产
2. 点击操作列的「删除」按钮
3. 系统弹出二次确认对话框
4. 点击「确认删除」完成操作

注意事项：

- 删除使用逻辑删除，数据不会真正从数据库移除
- 删除后该资产不再出现在资产列表中
- 删除操作会记录到操作日志

### 5.6 查看资产详情

操作步骤：

1. 在资产台账表格中点击操作列的「查看」按钮
2. 进入资产详情页面，展示资产的完整信息

详情页面包含：

- 基础信息：资产编号、资产名称、分类、品牌、规格型号
- 价值信息：原值、净值、累计折旧、残值率、使用年限、折旧方法
- 使用信息：购置日期、所属部门、使用人、存放地点
- 状态信息：资产状态（颜色标签）
- 标签信息：二维码、RFID
- 备注信息

## 6. 资产分类

### 6.1 功能介绍

查看固定资产分类体系，了解各分类的默认折旧年限。

### 6.2 查看方式

1. 点击左侧菜单「资产管理 > 资产分类」
2. 左侧展示分类树结构
3. 右侧展示分类列表表格
4. 表格包含：分类编码、分类名称、默认折旧年限、备注

---

## 7. 退出登录

点击顶部用户栏右侧的「退出登录」按钮，系统会清除登录信息并跳转至登录页。
